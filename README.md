## 开发计划

1. 场景渲染继续调整，天空盒优化。
2. 人物与场景碰撞/物理效果添加。
3. 故事内容等文案设计。
4. 追逐战等玩法添加，怪物模型设计等。
5. 跑步中的加速逻辑：主要玩法参考“~~8番出口~~（再想想）” 等游戏的找不同逻辑；在此基础上加入“通过拼命加速狂奔甩掉异常追逐，从而跳过该机制”的玩法。
6. 场景昼夜效果等添加。
7. 人物、场景素材替换。
8. 关卡设计，为场景添加新的解谜元素
9. 音效添加，音乐素材添加？？
